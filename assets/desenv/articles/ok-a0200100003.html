Como instalar o Angular

A instalação é através de um terminal de comandos, que poderá ser qualquer um de sua preferência.

No Windows você poderá utilizar o prompt de comando ou cmd, e para abrir a janela do terminal digite cmd na barra de pesquisa do Windows e clique para executar a aplicativo.

Utilizaremos o NPM para instalar o Angular, então faz-se necessário que você tenha instalado o NodeJS, sendo que o NPM faz parte do pacote de instalação do NodeJS.

Para verificar se o NPM está instalado digite no terminal o comando abaixo:

npm -v

Vamos então a instalação do Angular propriamente dito.

Caso você queira verificar se já tem alguma versão do angular instalada digite no terminal o comando abaixo:

ng --version

Finalmente, execute o comando abaixo para instalar o CLI do Angular em seu micro.

Esse comando irá instalar a última versão estável do CLI do Angular, e caso já tenha uma versão antiga instalada, a mesma será atualizada.

npm install -g @angular/cli

O Angular não tem uma interface ou tela, e sua utilização é através do CLI, ou Command Line Interface, ou ainda, Interface de Linha de Comando, que você utiliza digitando comandos no terminal, utilizando a abreviatura ng, que são interpretados pelo CLI e este executa as funcionalidades do framework, como nos exemplos abaixo:

>ng generate component para criar um novo componente.
>ng serve para executar o app localmente.

Execute novamente o comando ng --version para saber se a instalação/atualização ocorreu com sucesso, e se a versão for mostrada na tela, parabéns, agora é maõs a obra. bora desenvolver...
