Crie seu primeiro app Angular

Abaixo o passo a passo de como criar um app(ou aplicativo) Angular, o que é uma tarefa muito simples, sendo que com apenas um comando você já terá um aplicativo pronto para ser executado.

Abra um terminal de comandos de sua preferência, sendo que no Windows você poderá utilizar o prompt de comando ou cmd. Digite cmd da barra de pesquisa do Windows e clique para abrir o aplicativo.

Você já deverá ter o CLI do Angular instalado em seu computador.
Dica: digite ng no terminal e será mostrada uma lista de comandos do CLI.

No terminal de comandos, navegue até a pasta onde deseja criar seu projeto, e digite o comando abaixo:
Obs.: O nome do projeto nesse caso será app-exemplo, mas você poderá dar outro nome qualquer ao projeto.

ng new app-exemplo

Serão feitas algumas perguntas, que para esse primeiro projeto exemplo você poderá deixar as respostas default, teclando somente ENTER.

Após isso o CLI fará a baixa e configuração do novo projeto em uma nova pasta que será criada dentro da pasta onde você executou o comando de criação.

Essa nova pasta terá o mesmo nome do projeto.

Testando seu novo App

Para testar o aplicativo criado, entre na pasta do projeto e execute o comando ng serve conforme abaixo, que fará o build(compilação) do projeto e executará um servidor na porta 4200(padrão).

cd app-exemplo
ng serve -o

A opção -o do comando ng serve -o significa open, e faz com que seja aberto automaticamente o seu navegador padrão já com a página principal do seu projeto carregada.

Caso isso não aconteça, abra o navegador de sua preferência e digite a URL abaixo na barra de pesquisa:

https://localhost:4200/

Deverá ser mostrada uma página padrão de apresentação que o Angular gerá em todos os novos projetos.

Para "derrubar"(finalizar a execução) o Servidor, digite Controle + C no terminal de comandos, e confirme, mas isso não é necessário fazer, pode deixar o Servidor "rodando".

Alterando o seu projeto Angular

Para fazer alterações no projeto, utilize o editor de sua preferencia (eu utilizo o VSCode).

Abra o arquivo app.component.html que está na pasta src>app>.

Apague todo o código do arquivo, que foi incluído automaticamente pelo Angular, e inclua somente uma tag html com frase simples como a abaixo:

<span>Meu Hello Word.</span>

Salve os ajustes, e pronto, se o servidor estiver "rodando" você já verá a alteração refletida no navegador, alias, essa é uma excelente opção disponibilizada e que ajuda em muito na produtividade, pois você vai alterando o projeto e testando simultaneamente.
Obs.: O que acontece é que o servidor identifica que algum arquivo do projeto foi salvo e recarrega a página automaticamente no navegador.
