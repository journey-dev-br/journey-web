Exibição de data no formato DD/MM/AAAA

Se você simplesmente exibir uma variável Date sem qualquer formatação, ela será apresentada como vemos abaixo:

Para que essa mesma data seja exibida no formato DD/MM/AAAA existem muitas maneiras, uma delas é utilizar o Pipe date já existente no Angular, conforme no exemplo abaixo:

<span>{{ new Date() | date: 'dd/MM/yyyy'}}</span>

No exemplo acima, onde está o new Date() deverá ser colocada a variável que você deseja exibir.
